MAIVI Features

AI Text-to-Code Conversion

Our advanced natural language processing system converts plain text descriptions into executable Python code. Simply describe what you want to build, and our AI generates the code to create it.

  • Understands architectural terminology and concepts
  • Generates clean, optimized Python code
  • Supports complex parametric designs
  • Continuously learns from user feedback

Example:

Input:

"Create a curved wall that is 10 meters long and 3 meters high with a thickness of 0.3 meters. Add 5 rectangular windows evenly spaced along the wall."

Output:

import rhinoscriptsyntax as rs
import math

# Parameters
length = 10
height = 3
thickness = 0.3
num_windows = 5
window_width = 1.2
window_height = 1.5

# Create curved wall
radius = length / (2 * math.sin(math.pi/4))
arc = rs.AddArc(rs.WorldXYPlane(), radius, math.pi/2)
wall = rs.ExtrudeCurve(arc, rs.AddLine([0,0,0], [0,0,height]))
wall = rs.CapPlanarHoles(wall)

# Add windows
curve_params = rs.DivideCurve(arc, num_windows + 1, True, True)
for i in range(1, len(curve_params) - 1):
    param = curve_params[i]
    point = rs.EvaluateCurve(arc, param)
    normal = rs.CurveTangent(arc, param)
    normal = rs.VectorRotate(normal, 90, [0,0,1])
    window_plane = rs.PlaneFromNormal(
        [point[0], point[1], height/2], 
        normal)
    window = rs.AddRectangle(
        window_plane, 
        window_width, 
        window_height)
    rs.BooleanDifference(wall, 
        rs.ExtrudeCurve(window, 
        rs.AddLine([0,0,-thickness/2], 
        [0,0,thickness/2])))

Rhino 8 Integration

MAIVI seamlessly integrates with Rhino 8, executing the generated Python code directly within the Rhino environment through Grasshopper. This integration allows for immediate visualization and refinement of your architectural designs.

  • Direct execution in Rhino via Python scripting
  • Grasshopper component for visual programming
  • Real-time feedback and visualization
  • Support for all Rhino geometric operations

Revit Integration for BIM

Take your designs to the next level with our Revit integration. MAIVI automatically converts Rhino models into Revit-compatible BIM files, maintaining all parametric relationships and architectural information.

  • Automatic conversion to .bim format
  • Preservation of parametric relationships
  • Material and property mapping
  • BIM data enrichment for construction documentation
